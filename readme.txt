Projeto de Cadastro de Endereços

CRUD para endereços com acesso a um WEB Service para busca de CEP.

 - Utilizado a api Jersey para criação de um cliente que consome o serviço de busca de CEP.

 - Utilizado Spring MVC para a camada de controle.

 - Utilização de Spring para injeção de dependências.

 - Utilização da biblioteca json para trabalhar com JSON.

 - Utilização do padrão DAO para persistência;

 - Futuramente construir uma implementação do DAO que acesse um banco de dados via JPA.
